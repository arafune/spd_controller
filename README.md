# spd_controller

Script/Library to control the facilities in my work.

Following facilities are supported

- Texio
  - oscilloscope: GDS3500
- Keithley
  - DMM2700
  - K2000
  - K6514
- SIGMA Koki (SIGMA TECH)
  - GSC-02 (To control variable ND filter)
  - SC104 (SIGMA TECH)
- Thorlabs
  - K10CR1 (Rotation)
  - MFF101 (Flipper)
